# GitHub Setup Guide

## Step 1: Initialize Git Repository

```bash
cd "c:\Users\Giriraj\OneDrive\Desktop\MY NEW PROJECT ANTI WEB"
git init
```

## Step 2: Add All Files

```bash
git add .
```

## Step 3: Create Initial Commit

```bash
git commit -m "Initial commit: Anti-phishing app with frontend and backend"
```

## Step 4: Create GitHub Repository

1. Go to https://github.com/new
2. Create a new repository (e.g., "anti-phishing-app")
3. DO NOT initialize with README (you already have one)

## Step 5: Connect to GitHub

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

## Done! ✅

Your repository is now on GitHub with both frontend and backend.
