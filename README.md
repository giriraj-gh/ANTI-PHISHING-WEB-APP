# 🛡️ Anti-Phishing Application

A comprehensive, professional-grade anti-phishing detection platform with React frontend and Node.js backend.

## 📁 Project Structure

```
anti-phishing-app/
├── FRONTEND/frontend/    # React application
├── BACKEND/backend/      # Node.js API server
├── .gitignore           # Git ignore rules
└── README.md            # This file
```

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- MongoDB
- npm or yarn

### Backend Setup
```bash
cd BACKEND/backend
npm install
# Configure .env file
npm start
```

### Frontend Setup
```bash
cd FRONTEND/frontend
npm install
npm start
```

## 📖 Documentation

- Frontend: See `FRONTEND/README.md`
- Backend: See `BACKEND/backend/README.md`

## 🔐 Environment Variables

Create `.env` files in both frontend and backend directories. See respective README files for details.
